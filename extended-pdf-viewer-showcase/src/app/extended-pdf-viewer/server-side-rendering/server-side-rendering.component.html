Starting with version 16.2.14, ngx-extended-pdf-viewer supports server-side rendering out-of-the-box.

There's no live demo yet because it's not that easy to make the showcase support SSR. However, the presentations on my blog employ ngx-extended-pdf-viewer with server-side rendering. For example, have a look at the slides of my JavaLand talk.

Before version 16.2, ngx-extended-pdf-viewer used a special module to support server-side rendering. If you're still using it, update to ngx-extended-pdf-viewer 17+ and abandon NgxExtededPdfViewerServerModule in favor of the new, unified approach.
