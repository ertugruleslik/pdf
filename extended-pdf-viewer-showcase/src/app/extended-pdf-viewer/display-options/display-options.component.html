A few options allow you to influence the way the PDF file is rendered.

Caveat: Being able to modify [pageViewMode] has been introduced with version 17.3.0. So it's a pretty new feature. It's possible some errors slipped my attention yet. If so, please open a ticket.

[showBorders]="true"

0 (vertical) (default)
1 (horizontal)
2 (wrapped)
3 (single-page view)
off (default)
odd
even
single
book
multiple (default)
infinite scroll
Book mode and single-page mode don't look too impressive with the default zoom setting of 30%. However, this zoom setting enables you to see the effect of scrollMode="2" (i.e. wrapped).
